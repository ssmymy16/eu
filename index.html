Introduction

Computer architecture is the design of computer systems, including hardware components and their interconnections. It encompasses the way that computers are built and how they operate.

Central Processing Unit

The central processing unit (CPU) is the brain of the computer, responsible for performing arithmetic and logical operations. It is composed of the control unit and the arithmetic logic unit.

Memory

Memory refers to the various components of a computer that store data and instructions. The two main types of memory are random-access memory (RAM) and read-only memory (ROM).

Input/Output

The input/output (I/O) system of a computer is responsible for communicating with the outside world. This includes devices such as keyboards, mice, monitors, and printers.
